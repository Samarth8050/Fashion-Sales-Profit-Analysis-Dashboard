# 🗂 Dataset Description

The dataset contains **fashion retail sales transactions** including order details, product information, and revenue metrics.

### Key Dataset Fields

| Column           | Description                      |
| ---------------- | -------------------------------- |
| Order_ID         | Unique identifier for each order |
| Order_Date       | Date of purchase                 |
| Product_Category | Category of the fashion item     |
| Product_Name     | Name of the product              |
| Units_Sold       | Quantity sold                    |
| Unit_Price       | Price per item                   |
| Discount_%       | Discount applied                 |
| Sales_Amount     | Revenue generated                |
| Profit           | Profit from the order            |
| City             | Customer city                    |

The dataset was cleaned and processed before performing analysis.

# 🧹 Data Cleaning Process

The dataset was prepared using the following steps:

* Removed duplicate records
* Handled missing values
* Standardized city names
* Converted **Order_Date to date format**
* Calculated **Sales Amount = Units Sold × Unit Price**
* Created **Month column for trend analysis**
* Ensured correct data types for numerical columns

This ensured the dataset was **accurate, consistent, and analysis-ready**.
